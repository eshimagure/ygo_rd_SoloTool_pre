# 遊戯王ラッシュデュエル一人回しツール
公式アプリ「ニューロン」のデッキ画像からカードを読み込み、ブラウザ上で一人回し（プレイングの練習やシミュレーション）ができるWebアプリケーションです。

**[👉 実際のサイトを見る](https://eshimagure.github.io/ygo_rd_SoloTool_pre/)**

* **デッキ画像の自動読み込み (OCR)**
  ニューロンで出力したデッキ画像をアップロードします。
  OCRがメイン・EXの枚数を自動解析してカード画像を切り出します。
* **直感的なカード操作**
  ドラッグ＆ドロップでフィールド上の好きな位置へカードを配置できます。
* **表示形式の変更**
  カードの右クリック（スマホはダブルタップ）で表示形式を変更できます。
  攻撃表示・守備表示・裏側表示を切り替え可能です。
* **補助機能**
  サイコロやコイントスなどの補助機能を搭載しています。
  状態のローカル保存機能で途中からプレイを再開できます。

## 🛠 使用技術

* **Frontend**: React, TypeScript
* **Drag & Drop**: `@dnd-kit/core`, `@dnd-kit/sortable`
* **OCR**: `tesseract.js` (デッキ枚数の自動解析)
